# Portafolio con Astro

Proyecto de portafolio personal usando Astro y Bun como runtime/gestor de paquetes.

## Requisitos

- Bun instalado
- Node no es necesario para correr scripts del proyecto

## Primer arranque

Desde la raiz del proyecto:

```sh
bun install
bun run dev
```

Tu sitio quedara en:

- http://localhost:4321

## Scripts disponibles

```sh
bun run dev
bun run build
bun run preview
bun run astro -- --help
```
